# Norvin García - Portal Inmobiliario Premium

Aplicación web construida con React + Vite, TailwindCSS, Framer Motion y Leaflet.

## Scripts

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run preview`

## Stack

- React + Vite
- TailwindCSS
- Framer Motion
- Lucide React
- React Leaflet
- Preparado para Firebase
